# Aerocord
Browser-based chord instrument inspired by the Omnichord — play ambient pads, arpeggios and strum textures in real time using keyboard or hand tracking.
What is it?
Aerocord is a web instrument built around a chord system. You select a root note and a chord type, and the instrument responds across three layers simultaneously:

Pad — sustained ambient textures that bloom when you hold a chord
Strum — melodic string sounds you trigger by hovering over the strings
Auto-Arp — an arpeggiator that plays through the chord notes automatically

The chord system supports Major, Minor, Dom 7, Maj 7, Min 7, Diminished, Sus 4 and Add 9. All three layers always play in harmony with each other.
Sounds
Pads — Warm Glow, Deep Space, Angel Choir, Cathedral Organ, Membrane Drone, Hyper Pad, Tape Drone, DX Brass, Prepared Piano, Cello Swell
Strum — Glass String, Harp, Marimba, Bells, Koto, Vibraphone, Kalimba, Shakuhachi, Pluck Synth, Tibetan Bowl
How to play

Click a root note (C, D, E...) to set the chord
Hold modifier keys (shown on screen) to change the chord type
Hover over the strings to strum
Enable AUTO-ARP to let the arpeggiator play for you
Press SHOW MIXER to access sounds, FX and controls

Hand tracking via webcam is also supported — enable it from the interface.
Installation
No install required. Open the link and play.
To install as a desktop or mobile app, open the link in Chrome and click the install icon in the address bar.
